# Phase 2: 構成設計

## 前提条件

- `work/strategy.json` が存在すること（Phase 1 の出力）

## 目的

strategy.json を入力に、タイトル候補・説明文構成・チャプター構成を設計する。

## 手順

### Step 1: タイトル候補3案生成

各候補で異なるフックパターンを使用:

- 数字型: 「Claude Codeの設定5選」
- 結果型: 「開発速度が3倍になった方法」
- 疑問型: 「なぜAIエージェントは暴走するのか」
- 速報型: 「【速報】Claude新機能を徹底解説」

### Step 2: 説明文構成設計

```
1行目: 動画の核心（検索結果に表示される150文字）
2-3行目: 視聴者が得られる価値
チャプター: タイムスタンプ + キーワード含有タイトル
リンク: 関連リソース
ハッシュタグ: 3-5個
```

### Step 3: チャプター構成設計

- 3-7チャプター
- 各チャプターのキーポイント
- タイムスタンプの概算

### Step 4: 構成 JSON 出力

設計結果を `work/structure.json` に保存。

## 入力

- ファイル: `work/strategy.json`

## 出力

- ファイル: `work/structure.json`
- スキーマ: `schemas/structure.schema.json`

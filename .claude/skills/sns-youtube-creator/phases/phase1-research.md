# Phase 1: リサーチ & 戦略

## 前提条件

- 動画のトピック・内容説明が提供されていること

## 目的

YouTube 上の競合動画を調査し、キーワードトレンドを分析する。

## 手順

### Step 1: 競合動画調査

WebSearch で以下を調査:

- 検索キーワード: `{テーマ} site:youtube.com`
- 上位5-10動画のタイトル・再生数・チャンネル登録者数
- 人気動画の共通パターン（タイトル形式、サムネイル、尺）

### Step 2: キーワードトレンド分析

- テーマに関連する検索キーワードの傾向
- 日本語と英語の検索ボリュームの違い
- 直近のトレンドワード

### Step 3: 差別化ポイント決定

- 既存動画にない角度（最新情報、実践デモ、比較）
- カテゴリ決定: tutorial / review / demo / news
- 動画の尺の最適値を判定

### Step 4: 戦略 JSON 出力

調査結果を JSON オブジェクトとして出力する（ファイルへの保存は不要）。

## 入力

- ユーザーの動画トピック・内容説明

## 出力

- JSON オブジェクト（```json ブロックで出力）
- スキーマ: `schemas/strategy.schema.json` に準拠

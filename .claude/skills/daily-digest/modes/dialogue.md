# 掛け合いモード

さとる × まさお の対話形式で解説する動画モード。

## 発動条件

デフォルトモード。特に指定がない場合は掛け合いモードで実行。
ソロモードにするには「ソロ」「一人で」を明示する。

## キャラクター

### さとる（メイン解説）

- 役割: ニュース解説、考察、本音トーク
- 発話比率: 60%
- 詳細: @../shared/characters/satoru.md

### まさお（リアクション）

- 役割: 視聴者目線の質問、盛り上げ
- 発話比率: 40%
- 詳細: @../shared/characters/masao.md

## 対話パターン

```
さとる: トピック導入
まさお: リアクション
さとる: 解説
まさお: 確認「つまり〇〇ってこと？」
さとる: 補足 + 本音
まさお: 感想
...
```

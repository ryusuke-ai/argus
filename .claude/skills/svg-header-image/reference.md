# SVG Thumbnail Reference

## テーマ一覧

### v3 テーマ（リッチなエフェクト）

| テーマ     | 名前           | 特徴                       |
| ---------- | -------------- | -------------------------- |
| `midnight` | Midnight       | ダークブルー系、サイバー感 |
| `sunset`   | Sunset         | レッド/ピンク系、情熱的    |
| `forest`   | Forest         | グリーン系、自然・成長     |
| `ocean`    | Ocean Deep     | ブルー/シアン系、深海感    |
| `lavender` | Lavender Dream | パープル系、幻想的         |

| ----------- | -------------- | -------------------------------- |

## 出力SVG仕様

| 項目 | 値 |
|------|-----|
| ViewBox | 0 0 1920 1080 |
| Width | 1920px |
| Height | 1080px |
| 日本語フォント | 'Noto Sans JP', 'Hiragino Sans', 'Yu Gothic', sans-serif |
| 英語フォント | 'Inter', 'Helvetica Neue', Arial, sans-serif |

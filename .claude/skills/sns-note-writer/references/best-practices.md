# note ベストプラクティス

> 最終更新: 2026-02
> ソース: note公式ヘルプセンター、note株式会社プレスリリース、各種分析記事

## 1. プラットフォーム特性

### 基本情報

- クリエイター向け長文プラットフォーム（2014年サービス開始）
- 「スキ」（いいね）ベースのエンゲージメント
- 有料記事・定期購読マガジンで収益化可能
- 技術系よりも **体験談・考察・ストーリー** が好まれる
- ドメインパワー: 88.5（Ahrefs: 91）— 個人サイト平均30に対して圧倒的に高い
- 流入元の73%がnote内回遊と検索経由（2025年データ）
- Google検索経由の流入が増えたと回答したユーザー: 62.4%
- 法人利用: 50,000件以上（2025年6月時点）

### Zenn/Qiita との差別化

| 項目         | Zenn/Qiita                   | note                               |
| ------------ | ---------------------------- | ---------------------------------- |
| 主な読者     | エンジニア                   | 一般ユーザー・ビジネス層           |
| コンテンツ   | コード主体、技術的正確さ重視 | 体験談・考察・ストーリー重視       |
| 収益化       | Zennのみ（バッジ）           | 有料記事・マガジン・メンバーシップ |
| SEO          | ドメイン強い                 | ドメイン非常に強い（88.5）         |
| フォーマット | Markdown                     | リッチエディタ + 一部Markdown      |

同じ技術ネタでも、note では以下の視点で再構成する:

- 「なぜその技術を選んだか」（意思決定プロセス）
- 「開発の裏話」（チーム・個人のストーリー）
- 「学んだこと」（気づき・成長）
- 「失敗から得た教訓」（共感・学び）

## 2. アルゴリズムと発見性

### レコメンドエンジン（2026年2月刷新）

noteは2026年2月にレコメンドエンジンを全面刷新した。LLM（大規模言語モデル）を活用した新しいタグ付け技術を導入し、記事の内容を深く理解して適切なカテゴリ・トピックに自動分類する精度が大幅向上。

**刷新の効果（2026年1月、前年同月比）:**

- カテゴリページ内のインプレッション: **4.3倍**
- カテゴリページからのPV: **2.2倍**
- note内回遊: **1.5倍**
- 記事PV 3年間成長率: **3.2倍**

### アルゴリズムの評価指標

| 指標                 | 重要度               | 説明                                                     |
| -------------------- | -------------------- | -------------------------------------------------------- |
| 初速エンゲージメント | 最重要               | 投稿後1-2時間以内のスキ・コメント・シェア数              |
| 読了率               | 非常に高い           | 記事を最後まで読んだ割合。高いほど「質の高い記事」と判断 |
| 平均滞在時間         | 高い                 | 記事ページでの滞在時間                                   |
| 離脱率               | 高い（低いほど良い） | 途中で離脱した割合                                       |
| スキ数               | 高い                 | 累計のスキ数                                             |
| コメント数           | 中                   | 読者とのインタラクション                                 |
| フォロワー数         | 中                   | 多いと記事がレコメンドされやすい                         |

### 初速エンゲージメントの重要性

**投稿直後2時間の「スキ」が結果を左右する。** アルゴリズムは初速重視であり:

- 投稿後1-2時間のスキ・コメント・シェアが集中するほど「おすすめ」掲載されやすい
- フォロワーへの通知で初動のスキを獲得 → アルゴリズムが拡散を判断
- 最適な投稿時間帯に公開することが重要

### 読了率を高めるテクニック

- 冒頭で結論を示唆し、読者の興味を維持する
- H2/H3で区切り、スキャンしやすい構成にする
- 適度に画像・引用を挿入して視覚的に変化をつける
- 一文を短くし、テンポよく読める文体にする
- 2,000-5,000文字が最適。長すぎると離脱率が上がる

### 「おすすめ」「ピックアップ」の仕組み

| 種類             | 選定方法                  | 特徴                                                    |
| ---------------- | ------------------------- | ------------------------------------------------------- |
| おすすめ         | アルゴリズム + 編集部確認 | 高品質記事をAI+人力で選定。スキ数が少なくても選ばれうる |
| ピックアップ     | 編集部の手動選定          | 記事下部に「ピックアップされています」と表示            |
| カテゴリおすすめ | LLMベースの自動分類       | 2層構造（大カテゴリ→トピック）で整理                    |
| 今月のおすすめ   | 編集部キュレーション      | 月次で特に優れた記事を選定                              |

**編集部に選ばれるポイント:**

- 「役に立つ記事」を書くことが最重要基準（note運営公式見解）
- 記事構成の質（丁寧に書かれているか）
- 1,500文字以上が目安
- スキ数が一桁でも選ばれることがある（人間が読んで判断）

### SEO（検索エンジン最適化）

- noteのドメインパワー（88.5）を活かせば、個人ブログより検索上位に出やすい
- 2025年1月: Googleとの資本業務提携を発表（約4.89億円の資金調達）
- タイトルに検索キーワードを自然に含める
- 見出し（H2/H3）にもキーワードを配置
- 記事内にリンク（内部・外部）を適切に配置
- 定期的な更新がSEO上有利

## 3. コンテンツ方針

### 記事タイプ

| カテゴリ   | 説明                   | 例                                        |
| ---------- | ---------------------- | ----------------------------------------- |
| experience | 体験談・実践記録       | 「AIエージェントで自動化した1ヶ月の記録」 |
| insight    | 考察・意見             | 「なぜ Claude Code を選んだのか」         |
| tutorial   | やさしいチュートリアル | 「非エンジニアのための AI 活用入門」      |
| review     | レビュー・感想         | 「話題の AI ツールを全部試してみた」      |

### AI/テック系で伸びる記事パターン

1. **体験型レポート**: 「AIで○○してみた」系。具体的な結果と数値を含む
2. **Before/After 比較**: AI導入前後の変化を可視化
3. **初心者向け解説**: 専門用語を避け、身近な例えで説明
4. **ツール比較レビュー**: 複数ツールを公平に比較。実際に使った感想を添える
5. **失敗談からの学び**: 失敗をオープンにし、そこからの気づきを共有
6. **業界トレンド考察**: 最新動向に対する個人的な見解

**注意:** noteでは「AI・機械学習」カテゴリが設けられており、AIコンテンツへの受容性は高い。ただしAI生成丸投げ記事は読者に見抜かれやすく、個人の体験や視点を必ず含めること。

### 本文構成ガイドライン

1. **フック** — 読者の共感を得る問いかけや体験（冒頭3行で引き込む）
2. **背景** — なぜこれをやろうと思ったか（動機・文脈）
3. **本論** — 体験の詳細（時系列が効果的。具体的な数値や結果を含む）
4. **気づき** — やってみてわかったこと（意外性のある発見が読者を惹きつける）
5. **まとめ** — 読者へのメッセージ（次のアクションを促す）

### 品質基準

- 文字数: 2,000-5,000文字（最適ゾーン）
- 画像: アイキャッチ必須 + 本文中2-3枚
- H2/H3 で区切り、スキャンしやすい構成
- 個人的な視点・感情を入れる（「思った」「感じた」「驚いた」）
- 技術用語は平易に言い換える（専門家でない読者も理解できるように）
- 冒頭100文字で記事の価値を伝える（レコメンド時のプレビューに影響）

### タイトルの付け方

- 体験ベースの表現:「〜してみた」「〜した話」「〜してわかったこと」
- 数字を入れる:「3つの理由」「1ヶ月の記録」「5000文字で解説」
- 疑問形を使う:「なぜ〜なのか」「〜は本当か？」
- 感情を込める:「衝撃を受けた」「感動した」「後悔した」
- 100文字以内に収める

| 曜日    | 特徴                            | おすすめコンテンツ |
| ------- | ------------------------------- | ------------------ |
| 月曜    | 週始め、モチベーション高い      | ビジネス・自己啓発 |

### 投稿頻度の目安

- 週1-2回の定期投稿が理想（質を維持できる頻度）
- 曜日・時間を固定するとフォロワーが習慣化しやすい
- 毎日投稿よりも **質の高い週1投稿** の方がアルゴリズム的に有利
- 投稿間隔が2週間以上空くとフォロワーの関心が薄れる

## 5. タグ・マガジン戦略

### タグの活用

- 最大10個まで設定可能
- **大タグ（安定）+ 小タグ（トレンド）** の組み合わせが効果的
  - 例:「#エッセイ #生成AI」「#日常 #AIエージェント」
- 公式お題タグへの参加でピックアップ対象になりやすい
- 日本語タグがnoteでは有効（英語タグよりリーチが広い）
- 人気タグの例: #毎日note, #エッセイ, #日記, #AI, #仕事, #暮らし

### タグ選定のコツ

1. **必ず入れるべきタグ**: 記事のメインテーマ（例: #AI, #プログラミング）
2. **発見性を高めるタグ**: 読者の関心事（例: #働き方, #副業）
3. **トレンドタグ**: 今話題のキーワード（例: #Claude, #生成AI）
4. **公式お題タグ**: noteが定期開催するお題企画のハッシュタグ
5. **カテゴリタグ**: noteのカテゴリ分類に合致するタグ

### マガジン活用

- テーマ別に記事をまとめることで読者の回遊を促進
- マガジンフォロワーが増えると新着通知で読まれやすくなる
- 有料マガジン（定期購読）は安定収入源になる
- 1マガジン5-10記事を目安に、テーマを絞って構成

### マガジン構成の例（AI/テック系）

```
[無料マガジン]
- AI活用の日常記録（体験談メイン）
- テックツールレビュー集

[有料マガジン]（月額500-1,000円）
- AI実践テクニック集
- 開発プロジェクトの裏側
```

## 6. 収益化戦略

### 有料記事の価格帯

| 価格帯        | 用途           | 適したコンテンツ                        |
| ------------- | -------------- | --------------------------------------- |
| 100-300円     | お試し・導入   | 短めのTips、テンプレート単体            |
| 300-500円     | カジュアル     | 副業ノウハウ、日常で使えるTips          |
| 500-800円     | **初心者推奨** | 体験談+具体的な手順。「気軽な自己投資」 |
| 800-1,000円   | 中級           | テンプレート付き解説、AIプロンプト集    |
| 1,500-2,500円 | 上級           | 特化スキル講座、学習型コンテンツ        |
| 3,000円以上   | プレミアム     | 独自ノウハウ、コンサル的要素            |

- 最低価格: 100円
- 最高価格: 一般50,000円 / プレミアム会員・note pro 100,000円

### 前半無料 + 後半有料（推奨パターン）

```
[無料部分]
- 導入: 問題提起・共感（読者の痛みに寄り添う）
- 概要: 何をしたか（結論の一部を見せる）
- ハイライト: 最も面白い結果（購入意欲を刺激）

[有料部分]（500-800円が初心者向け推奨）
- 詳細な手順
- 具体的な数値・データ
- テンプレート・チェックリスト
- 失敗パターンと対策
```

**有料ライン設置のコツ:**

- 無料部分だけでも価値があるように構成する（信頼構築）
- 有料部分に「これだけ払う価値がある」と思わせる具体性を入れる
- 無料部分で結論の一部を見せ、「もっと知りたい」と思わせる

### 手数料体系

| 項目                           | 料率                               |
| ------------------------------ | ---------------------------------- |
| プラットフォーム利用料         | 売上の10%（定期購読マガジンは20%） |
| 決済手数料（クレジットカード） | 5%                                 |
| 決済手数料（携帯キャリア）     | 15%                                |
| 決済手数料（PayPay）           | 7%                                 |
| 振込手数料                     | 270円/回                           |

**実質手取り: 売上の約75-85%**（決済方法により変動）

例: 1,200円の記事（クレジットカード決済）

- 決済手数料: 60円（5%）
- プラットフォーム利用料: 114円（残額の10%）
- 手取り: 1,026円（振込手数料270円控除前）

### 松竹梅の法則

異なる価格帯の商品を複数用意すると、中間価格帯が選ばれやすい:

- **梅（お試し）**: 500円 — 単発Tips
- **竹（標準）**: 1,500円/月 — 定期購読マガジン
- **松（高付加価値）**: 5,000円 — 個別サポート付きコンテンツ

### 価格調整の戦略

- 初期は控えめに設定し、読者の反応を観察
- 販売実績が積み上がったら徐々に値上げを検討
- 同テーマの人気記事の価格帯をリサーチして相場感を把握
- セール（期間限定値下げ）は新規読者獲得に効果的

## 7. 禁止事項と注意点

### noteの利用規約・コミュニティガイドライン

**絶対にやってはいけないこと:**

1. **著作権侵害**: 他者の文章・画像の無断転載、盗作、剽窃
2. **情報商材**: マルチ商法等、ユーザーに不利益をもたらす情報商材
3. **金融商品推奨**: 株式銘柄推奨など金融商品取引法に抵触する内容
4. **誇大広告・虚偽表現**: 実績の水増し、効果の保証
5. **差別的表現**: 民族・宗教・人種・性別・年齢等に関する差別
6. **危険なコンテンツ**: 自殺・自傷・違法薬物使用の勧誘・助長
7. **わいせつ・暴力的表現**: 過度な不快感を与える内容

**ペナルティ:**

- 記事の公開停止
- アカウント停止（BANされると全記事が消える）
- **売上金の没収**（未振込分が全額没収される可能性）
- 違約金請求

### コンテンツ品質に関する注意

- AI生成記事の丸投げは推奨されない（読者に見抜かれやすく、スキも付きにくい）
- 「いかがでしたか」系の定型文は避ける
- 感情のない事務的な文章は note の読者層に合わない
- コードだらけの記事は Zenn/Qiita に任せる
- 他プラットフォームのコピペ転載は評価が下がる

### API・自動化の制約

- **noteは公式投稿APIを提供していない**（将来の公開予定も未定）
- 非公式APIは存在するが、規約違反リスクあり。仕様変更で突然動かなくなる可能性
- RSSフィードは公式に利用可能（記事の外部通知やSNS連携に活用）
- Seleniumなどの自動化ツールによる投稿は非推奨
- 現実的な自動化: Markdown で記事を生成 → 手動でnoteに貼り付け
- SNS連携の自動化（note投稿 → X自動ポスト）は Zapier + Buffer 等で可能

### note pro（法人向け）

- 月額80,000円（税抜）
- 独自ドメイン設定
- ロゴ・テーマカラーのカスタマイズ
- note pro アナリティクスβ（読了率などの詳細分析）
- AIアシスタント機能（執筆サポート）
- 予約投稿機能
- サイト作成機能（最大3ページ、6種テンプレート）
- カスタマーサクセスによる個別サポート
- 最高価格100,000円の有料記事設定が可能

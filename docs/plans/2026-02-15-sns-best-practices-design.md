# SNS全媒体ベストプラクティス拡充 設計書

> 作成日: 2026-02-15

## 目的

X (Twitter) のみ充実していたベストプラクティスドキュメントを全10媒体に展開し、投稿品質を均一に向上させる。

## スコープ

### A: best-practices.md 作成/拡充

| 媒体      | 作業               | 目標行数 |
| --------- | ------------------ | -------- |
| Threads   | 新規作成           | 300行+   |
| Instagram | 新規作成           | 300行+   |
| TikTok    | 新規作成           | 300行+   |
| YouTube   | 新規作成           | 300行+   |
| GitHub    | 新規作成           | 300行+   |
| Podcast   | 新規作成           | 300行+   |
| Qiita     | 拡充 (74行→300行+) | 300行+   |
| Zenn      | 拡充 (65行→300行+) | 300行+   |
| note      | 拡充 (72行→300行+) | 300行+   |

### B: プロンプト反映

各媒体の `prompts/*.md` と `phases/*.md` にベストプラクティスの要点を組み込む。

### C: validator.ts 強化

調査結果に基づいてバリデーションルールを追加・更新。

### D: optimal-time.ts 見直し

調査結果に基づいて最適投稿時間を更新。

## 実行計画

- Batch 1: Threads, Instagram, TikTok（SNS短文・動画系）
- Batch 2: YouTube, Podcast, GitHub（動画・音声・リポジトリ系）
- Batch 3: Qiita, Zenn, note（記事系、既存拡充）
- Batch 4: validator.ts + optimal-time.ts（横断的更新）

## 調査方法

各媒体について2025-2026年の最新情報をWeb調査し、ソースURL付きで記載。
